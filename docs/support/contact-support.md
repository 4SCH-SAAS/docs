# Contact & Support

- School Admins: Contact your 4SCH success manager or submit a ticket via Admin Console → Support
- Teachers/Parents/Students: Reach out to the school office first; they can escalate with context
- Critical outages: Use the emergency contact listed in Admin Console → Support → On-call

Appendix: Example CSV headers
Use the built-in templates from Admin Console → Import. Typical headers:
- Students: student_id, first_name, last_name, class, section, date_of_birth, guardian_email
- Staff: staff_id, first_name, last_name, email, role, subject
- Subjects: subject_code, subject_name, class, elective
- Enrollments: student_id, class, section, session, term
