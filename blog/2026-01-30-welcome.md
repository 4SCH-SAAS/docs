# Welcome to 4SCH Release Notes

We're excited to launch our new blog for release notes and updates! This is where we'll share:

- **New Features**: Exciting new capabilities added to the platform
- **Improvements**: Enhancements to existing functionality
- **Bug Fixes**: Issues resolved in each release
- **Important Announcements**: System updates, maintenance windows, and more

## What's New in This Release

### Search Functionality
We've added Algolia DocSearch to make it easier to find what you need in our documentation. Simply use the search bar in the navigation to quickly locate topics, features, and solutions.

### Documentation Versioning
You can now access documentation for different versions of 4SCH. Use the version dropdown in the navigation to switch between versions.

### Release Notes Blog
Stay informed with our new blog dedicated to release notes and product updates. Subscribe to our RSS feed to never miss an update!

## Stay Connected

We'll be posting regular updates here, so bookmark this page and check back often. If you have questions about any release, please [contact our support team](/support/contact-support).
